Requests for Comments (RFCs)

Neurokernel's long-term development planning is conducted via the composition and revision of Requests for Comments, or RFCs. First employed in the development of the Internet, RFCs are publicly accessible design proposals for major new components or features of a complex computing system. Anyone can write and submit a Neurokernel RFC for consideration; see this page on the project wiki for more information.

Neurokernel RFC Information
#1 Neurokernel: An Open Scalable Software Framework for Emulation and Validation of Drosophila Brain Models on Multiple GPUs, L.E. Givon, A.A. Lazar, November 2013
#2 The Cartridge: A Canonical Neural Circuit Abstraction of the Lamina Neuropil - Construction and Composition Rules, A.A. Lazar, N.H. Ukani and Y. Zhou, January 2014

IPython Notebooks

Demonstrations of how to use Neurokernel are available as notebooks for the IPython interactive computing environment. All notebooks can be downloaded as part of Neurokernel's source code.

Demo Information
Introduction Introduction to using Neurokernel's API.
Mapping a Network of LPUs onto Multiple GPUs How to connect and execute several generic LPUs on multiple GPUs.
Olfaction Depicts spikes generated by partial Drosophila antennal lobe model in response to a basic olfactory concentration stimulus.
Vision Depicts membrane potentials of graded potential neurons in partial Drosophila vision system model generated in response to a basic visual input stimulus.
Sensory Integration Depicts an example of how to integrate models of Drosophila's olfactory and vision systems into a model of multisensory coincidence detection.
Lamina Configuration Depicts the configuration of Drosophila's Lamina model described in Neurokernel RFC #2.
