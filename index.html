What Is It?

The Neurokernel project aims to build an open software platform for the emulation of the entire brain of the fruit fly Drosophila melanogaster on multiple Graphics Processing Units (GPUs). Read more here.

How Can I Try It?

Interested? Great! Clone the source from GitHub, join the project mailing list, and start submitting pull requests and bug reports.
