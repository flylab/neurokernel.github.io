Neurokernel

An open source platform for emulating the fruit fly brain

From in vivo to in silico

Modeling the fly brain

Developed in Python and CUDA

IPython Notebook demos

What Is It?

The Neurokernel Project aims to build an open software platform for the emulation of the entire brain of the fruit fly Drosophila melanogaster on multiple Graphics Processing Units (GPUs).

How Can I Try It?

Interested? Great! Clone the source from GitHub, join the project mailing list, and start submitting pull requests and bug reports.
